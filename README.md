## 📚 Acerca del Curso

**Iron Man Evolution** es un curso donde **Tony Stark te guía desde cero absoluto hasta orchestrar un sistema multi-agente de desarrollo**.

Comenzamos en una cueva en Afghanistan, sin recursos, y construimos hasta el Nanotech de Avengers: Infinity War.

**El enfoque es progresivo**: cada nivel introduce conceptos que se construyen sobre los anteriores, usando la narrativa de las armaduras de Iron Man.

## 🧠 Principios Arquitectónicos

El curso implementa los principios del sistema multi-agente de Gentleman Programming:

| Principio | Descripción |
|-----------|-------------|
| **Security First** | Seguridad como requisito, no addon |
| **Zero Trust** | Todo input es potencialmente malicioso |
| **Spec-Driven Development** | El código es subproducto de especificaciones |
| **Aislamiento Multi-Agente** | Git Worktrees para trabajo aislado |
| **Flujo de 7 Pasos** | ANALYZE → PROPOSE → REVIEW → DESIGN → APPLY → ARCHIVE → ITERATE |

El curso está disponible en GitHub Pages:

👉 **[https://statick88.github.io/iron-man-evolution/](https://statick88.github.io/iron-man-evolution/)**

### Local

```bash
# Clonar el repositorio
git clone https://github.com/statick88/iron-man-evolution.git
cd iron-man-evolution

# Instalar Quarto (si no lo tienes)
brew install quarto  # macOS
# o descargar de https://quarto.org

# Renderizar localmente

```
