8.5 🔧 Configuración Recomendada por Uso

8.5.1 Setup Mínimo (Gratis)

# 1. Gemini CLI - Para empezar
npm install -g @google/gemini-cli

# 2. Amp - Para tareas rápidas
npm install -g @anthropic-ai/amp

# Costo total: $0

8.5.2 Setup Profesional (~$10/mes)

# 1. OpenCode - Tu agente principal
curl -fsSL https://opencode.ai/install | bash

# 2. API Keys necesarias
# - Anthropic: $5/mes (crédito inicial gratis)
# - OpenAI: $5/mes (crédito inicial gratis)

# Costo total: ~$10/mes

8.5.3 Setup Avanzado (~$50/mes)

# 1. OpenCode - Flexibilidad
# 2. Claude Code - Potencia
npm install -g @anthropic-ai/claude-code

# 3. KiloCode - Orquestación
code --install-extension kilo-code.kilo-code

# Costo total: ~$50/mes

8.5.4 Setup Enterprise (~$200/mes)

# Stack completo
# - Claude Code (Opus)
# - Kiro
# - KiloCode Pro
# - OpenCode enterprise

# Costo total: ~$200/mes

8.10 ❓ Preguntas Frecuentes

8.10.1 ¿Puedo usar múltiples herramientas a la vez?

Sí. Cada herramienta tiene su strengths. Combiná según necesidad:

  • Claude Code para debugging complejo
  • OpenCode para flexibilidad
  • Amp para tareas rápidas
  • KiloCode para multi-agente

8.10.2 ¿Cuánto cuesta usar estas herramientas?

Herramienta Costo Real
Gemini CLI $0 (límites gratis)
Amp $0 + API (~$5/mes)
OpenCode $0 + API (~$10/mes)
Claude Code $20-200/mes
KiloCode $19/mes + API
Kiro $20-200/mes

8.10.3 ¿Cuál es la mejor para principiantes?

Gemini CLI. Cero configuración, gratis, 1M tokens.

8.10.4 ¿Cuál es la mejor para producción?

Claude Code + OpenCode. Combinas potencia con flexibilidad.

8.10.5 ¿Necesito todas?

No. Empezá con una, dominála, después expandí.

## ❓ Preguntas Frecuentes

### ¿Puedo usar múltiples herramientas a la vez?

**Sí.** Cada herramienta tiene su strengths. Combiná según necesidad:

- Claude Code para debugging complejo
- OpenCode para flexibilidad
- Amp para tareas rápidas
- KiloCode para multi-agente

### ¿Cuánto cuesta usar estas herramientas?

| Herramienta | Costo Real |
|-------------|------------|
| Gemini CLI | $0 (límites gratis) |
| Amp | $0 + API (~$5/mes) |
| OpenCode | $0 + API (~$10/mes) |
| Claude Code | $20-200/mes |
| KiloCode | $19/mes + API |
| Kiro | $20-200/mes |

### ¿Cuál es la mejor para principiantes?

**Gemini CLI.** Cero configuración, gratis, 1M tokens.

### ¿Cuál es la mejor para producción?

**Claude Code + OpenCode.** Combinas potencia con flexibilidad.

### ¿Necesito todas?

**No.** Empezá con una, dominála, después expandí.
